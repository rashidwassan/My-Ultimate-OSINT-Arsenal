Open-Source Intelligence (OSINT) is a discipline of gathering publicly available data from online sources to uncover valuable insights. This repository is your ultimate guide to tools, training, and techniques that will help you master OSINT.

## üõ†Ô∏è Working with OSINT Tools
The tools listed here are essential for gathering information during OSINT investigations. - **Amass**: [Comprehensive Subdomain Enumeration](https://github.com/OWASP/Amass)
- **GoWitness**: [Take Website Screenshots](https://github.com/sensepost/gowitness/wiki/Installation)
